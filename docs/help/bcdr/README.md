# BC/DR Help Documentation

Business Continuity and Disaster Recovery (BC/DR) module documentation.

## Core Features

- [Dashboard](dashboard.md) - BC/DR overview and metrics
- [Business Processes](business-processes.md) - BIA and process management

- [DR Tests](dr-tests.md) - Disaster recovery testing

## Enhanced Features

- [BIA Wizard](bia-wizard.md) - Guided business impact analysis questionnaire

## Quick Start

1. **Assess Your Processes**: Use the [BIA Wizard](bia-wizard.md) to identify critical business processes
2. **Create Recovery Plans**: Document recovery procedures in [BC/DR Plans](plans.md)
3. **Define Teams**: Set up [Recovery Teams](recovery-teams.md) for incident response
4. **Test Your Plans**: Schedule [DR Tests](dr-tests.md) using [Exercise Templates](exercise-templates.md)
5. **Track Vendor Gaps**: Monitor [Vendor Dependencies](vendor-dependencies.md) for recovery capability gaps
6. **Manage Incidents**: Use [Incidents](incidents.md) for real events or drills
