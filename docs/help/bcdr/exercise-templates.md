# Exercise Template Library

The Exercise Template Library provides pre-built tabletop exercise scenarios for DR testing and training.

## Overview

Exercise templates help you:
- Run realistic tabletop exercises without extensive preparation
- Use industry-standard scenarios covering common disruption types
- Maintain consistency across multiple exercises
- Save time creating exercise content

## Accessing the Library

1. Navigate to **BC/DR -> Exercise Templates**
2. Browse available templates
3. Filter by category or search by keyword

## Available Template Categories

| Category | Description |
|----------|-------------|
| **Ransomware** | Cyber attacks encrypting systems and demanding payment |
| **Infrastructure** | Data center, cloud, or critical system failures |
| **Vendor Outage** | Critical third-party service disruptions |
| **Natural Disaster** | Facility inaccessibility due to weather or natural events |
| **Pandemic** | Workforce unavailability and remote work scenarios |
| **Data Breach** | Unauthorized access to customer or sensitive data |

## Built-in Global Templates

The library includes 6 pre-built global templates:

1. **Ransomware Attack** - Simulated ransomware affecting critical systems
2. **Data Center / Cloud Failure** - Major cloud provider outage
3. **Critical Vendor Outage** - Key SaaS provider disruption
4. **Natural Disaster** - Facility inaccessibility scenario
5. **Pandemic Response** - Extended remote work and workforce disruption
6. **Data Breach** - Customer data exposure and breach response

## Template Contents

Each template includes:

### Scenario Narrative
A detailed description of the incident that sets the scene for participants.

### Discussion Questions
Structured questions to guide the exercise, organized by category:
- Initial Response
- Communication
- Decision Making
- Recovery
- Compliance

### Scenario Injects
Timed updates during the exercise that introduce new developments:
- Media inquiries
- Expanded scope discoveries
- Vendor communications
- Customer escalations

### Participant Roles
Recommended roles for the exercise:
- Executive Sponsor
- IT Lead
- Legal Counsel
- Communications/PR
- Business Unit Leaders

### Facilitator Notes
Guidance for the person running the exercise on what to observe and explore.

## Using a Template

1. Browse the template library
2. Click on a template to preview its contents
3. Click **Use This Template**
4. The template will be cloned to your organization
5. A new DR Test will be created with the template content pre-filled
6. Customize as needed and schedule the exercise

## Creating Custom Templates

1. Navigate to **BC/DR -> Exercise Templates**
2. Click **Create Custom Template**
3. Fill in the template details:
   - Title and description
   - Category and scenario type
   - Scenario narrative
   - Discussion questions
   - Optional injects and facilitator notes
4. Click **Create Template**

### From a Completed Test

After completing a successful DR test, you can save it as a template:

1. Open the completed DR test
2. Click **Save as Template**
3. Provide a title for the template
4. The test scenario and learnings become a reusable template

## Template Types

| Type | Description |
|------|-------------|
| **Tabletop** | Discussion-based, no actual system changes |
| **Walkthrough** | Step-by-step procedure review |
| **Simulation** | Full simulation with actual system changes |

## Global vs. Organization Templates

- **Global Templates**: Pre-built templates available to all organizations. Cannot be modified.
- **Organization Templates**: Templates you create or clone. Fully editable.

When you clone a global template, it becomes an organization template that you can modify.

## Usage Tracking

Each template tracks its usage count, helping you identify popular and effective scenarios.
